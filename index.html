Template examples:
I can
  • do a lot of javascripty things {{5*2}}
  • do ternary operations {{'foo' === 'foo' ? 'foo' : 'bar'}}
  • do ternary operations on attributes
  • use filters for data display (more on that later)
  • interact with variables on the scope (more on that later)
I can't
  • access window: {{window.alert('Hey!')}}
  • create functions
  • do flow control stuff (like loops, and if statements)
  • do regex
I don't
  • throw errors with undefined or null: {{undefined() && null['sweet!']}}
